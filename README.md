# Hora_do_dia
- Mini programa que mostra na tela o tempo (hora) em tempo real, enquanto muda de imagem e tela de fundo (manhã, tarde e noite);
- Feito em HTML, CSS e JS;
- Utiliza a dinâmica do DOM + functions;
